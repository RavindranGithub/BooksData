The project is created in Asp.net core

Models and controllers are in the respective folders of same name.

DB script and backup are included in th Resources Folder

The Text file BulkInsert Books.txt has the sample book data
--------------------------------------------------------------------
Apis:

Api to get list of all books
api/Books

Api to get list of all books sorted by publisher
api/Books/sortedbypublisher

Api to get list of all books sorted by author
api/Books/sortedbyauthor

Api to get total price of all books
api/Books/totalprice

Api to post book data
api/Books/bulk-insert

Api to get list of all books sorted by publisher using stored procedure
/api/Books/sortedbypublisherwithsp

Api to get list of all books sorted by author using stored procedure
/api/Books/sortedbyauthorwithsp
